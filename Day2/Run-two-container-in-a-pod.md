## Hands-on steps:
```
Step 1: Open VS code and create new folder <br />
Step 2 : write manifest file pod2.yml, copy code from below link<br />
https://github.com/arjunachari12/k8s-learn/blob/main/pod2.yml<br /> 
Step 3: Execute pod2.yml file using below command<br />
kubectl create -f pod2.yml<br />
Step 4: You should see pod created message<br />
Step 5: run below commands<br />
kubectl get pod<br />
kubeclt describe pod pod2<br />
kubectl delete pod pod2<br />
```
